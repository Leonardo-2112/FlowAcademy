# 📖 Sobre o projeto

O **FlowAcademy** é um sistema integrado de gestão acadêmica desenvolvido como Projeto Integrador do curso Técnico em Tecnologia da Informação do **Senac**.

O projeto surgiu para solucionar um problema comum em instituições de ensino: a descentralização das informações acadêmicas, administrativas e financeiras. Em muitas escolas esses processos são realizados em sistemas diferentes ou até manualmente, dificultando o acompanhamento dos alunos e aumentando o tempo gasto nas rotinas administrativas.

Pensando nisso, foi desenvolvida uma plataforma única capaz de centralizar todas essas informações, proporcionando maior organização, agilidade e segurança para alunos, professores e colaboradores da instituição.

Além do gerenciamento acadêmico, o sistema também permite o acompanhamento financeiro, controle de matrículas e acesso personalizado de acordo com o perfil de cada usuário.

---

# 🎯 Objetivos

- Centralizar os processos acadêmicos, administrativos e financeiros.
- Facilitar o gerenciamento da instituição.
- Melhorar o acompanhamento pedagógico dos alunos.
- Organizar notas, frequência e matrículas.
- Controlar pagamentos e informações financeiras.
- Disponibilizar diferentes níveis de acesso conforme o perfil do usuário.
- Aplicar na prática os conhecimentos adquiridos durante o curso.
